# Tic Tac Toe

A console-based Tic Tac Toe game for two players.

## Features
- 3x3 board
- Player X and O take turns
- Win detection (rows, columns, diagonals)
- Draw detection

## How to Run
```bash
g++ main.cpp -o tic_tac_toe
./tic_tac_toe
